This is a code for for Discord bot which uses [https://nyos.dev/prescript](https://nyos.dev/prescript) to take prescriptions and send them in text channels on discord via command **!prescript**.

To launch:

1) Create venv and activate it:
```powershell
python -m venv venv
venv/Scripts/activate
```
2) Install dependencies:
```powershell
pip install -r requirements.txt
```
3) Set your bot's token in .env.template and **remove .template part**
4) Launch:
```powershell
python main.py
```
